## ss-backend API

This API primarily validates and applies a coupon to a cart

[link to live heroku](https://shubham-ss-api.herokuapp.com/api/)

###Routes

/api/{type}

/api/cart/{userId}

/api/cart/{userId}/{task}

/api/coupon/{id}

/api/validate/{userId}/{couponId}

/api/apply/{userId}/{couponId}
